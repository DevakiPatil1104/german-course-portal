# German Course Portal
## Overview

A clean, responsive, and professional web portal designed to showcase German language courses and facilitate student inquiries.

## Live Demo
https://german-course-portal.vercel.app/

## About the Project

This project serves as a centralized platform for "German with Srushti." It allows prospective students to browse available courses, understand the curriculum, and easily contact the instructor for enrollment details.

## Tech Stack 

Frontend: React.js

Build Tool: Vite

Styling: Tailwind CSS

Deployment: Vercel

## Features

Course Catalog: Detailed overview of course offerings.

Mobile-Responsive Design: Seamless experience across desktop, tablet, and mobile devices.

Quick Contact: Integrated WhatsApp link for immediate communication.

Clean UI/UX: Professional layout focused on clarity and ease of use.

## Future Improvements

User Authentication: Implementing secure login/signup functionality for students to track their course progress.

Payment Integration: Adding a gateway to handle course registrations and payments directly through the portal.
